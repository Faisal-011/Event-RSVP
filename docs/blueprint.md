# **App Name**: Eventide RSVP

## Core Features:

- Form Submission: Allow users to submit RSVP information via a form, including name, email, and any special requests.
- Data Persistence (Supabase): Persist user-submitted RSVP data to a Supabase NoSQL database, with appropriate schema design to accommodate user inputs.
- Data Retrieval: Retrieve specific RSVP records from the Supabase database using the Supabase API, for example using the email or user ID
- Record Display: Display the retrieved RSVP record in a clean, readable format, ensuring sensitive information is handled securely.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calming and minimalist feel.
- Background color: Off-white (#F8F8FF) for a clean backdrop.
- Accent color: Muted purple (#A9A9D1) for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look.
- Use simple, outline-style icons for clarity.
- Emphasize clean, whitespace-rich layouts to highlight content.
- Incorporate subtle transitions and animations to enhance user experience.
